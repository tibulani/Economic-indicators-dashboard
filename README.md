# Data-Driven Economic Indicators Dashboard

A full-stack project that fetches, processes, and visualises key economic indicators including GDP, Inflation, and Unemployment.

## Project Overview
This dashboard demonstrates the practical application of econometrics knowledge using software tools. It consumes API data using JavaScript and processes datasets using Python to ensure data accuracy, validation, and structured presentation.

## Features
- Fetches real economic data from public APIs
- Data cleaning and validation using Python (Pandas)
- Interactive dashboard built with HTML, CSS, and JavaScript
- Visual representation of GDP, Inflation, and Unemployment trends
- Focus on data accuracy and structured reporting

## Technologies Used
- HTML5, CSS3, JavaScript
- Python (Pandas)
- REST APIs
- Git & GitHub

## How to Run
1. Run `data_processing.py` to clean and prepare the dataset
2. Open `index.html` in your browser
